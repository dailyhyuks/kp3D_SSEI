# KP3D v2 설계: 구륵법 인지 선·색 분해 백본 (Line–Color Decomposition Backbone)

- 날짜: 2026-07-05
- 상태: 설계 확정 (사용자 승인 완료, 섹션별 검토)
- 대상: 차기 논문(v2) — 현재 논문(v1)의 후속 연구

---

## 0. 요구사항 (인터뷰 결과)

| 항목 | 결정 |
|---|---|
| 목적 | 후속 연구/차기 논문의 이상적 설계 (v1 코드 교체 아님) |
| P2 원칙 | **Training-free 유지** (사전학습 모델 추론은 허용, 학습·fine-tuning 불허) |
| 최종 출력 | 객체별 360° 메시 (v1과 동일) |
| 핵심 기여 | **Stage 1 직물 제거 품질 심화 + Stage 3 인페인팅 심화** |
| Stage 1 범위 | 직물 제거 품질 심화 (손상 유형 확장 아님) |
| 평가 | **정량 평가 위주** (전문가 정성 평가 배제, 합성 데이터 GT 실험 허용) |

## 핵심 아이디어

한국화가 구륵법(윤곽선 + 채색)으로 그려진다는 도메인 지식을 시스템 전체의 공통 표현으로 삼는다. 파이프라인 맨 앞에서 그림을 **선 레이어 L + 색 레이어 C**로 분해하면:

- Stage 1: 직물 주파수와 먹선 고주파의 스펙트럼 겹침(v1의 근본 딜레마)이 소멸 → 타협 장치(α_min, 보호 대역 등) 제거
- Stage 3: 인페인팅이 "선 완성 + 영역별 색 채움"으로 분해 → 어려운 문제 하나가 쉬운 문제 여러 개로

하나의 개념이 두 핵심 기여를 관통하여 논문 정체성을 형성한다.

## 설계 원칙

- **P1 모듈성, P2 training-free, P3 객체별 처리**: v1에서 승계
- **P-adapt (신규)**: 모든 파라미터는 ① 이미지 통계에서 유도(직조 주기, 선폭 분포, 대비 분포, 노이즈 바닥 σ_n)하거나 ② 서수 비교(argmax 자가 경쟁)로 대체한다. 수학적 유도 상수 외 튜닝 상수 0개.

## 1. Stage 0 (신규): 선·색 분해

### 1.1 파이프라인

```
원본 I
 ① Rolling Guidance Filter (σ_s = 자동 검출된 직조 주기)
    → 소규모 텍스처 제거된 구조 이미지 S (강한 먹선 보존)
 ② S에서 scale-space 다중 σ DoG + 적응 임계화 → 선 후보
    → 스켈레톤 + 폭 측정 (v1 measure_edge_width_map 로직 재사용)
 ③ 분리:
    L (선 레이어) = 선 마스크 영역의 원본 픽셀 + soft alpha (선 강도 기반)
    C (색 레이어) = 원본에서 선을 Telea fast inpainting으로 제거
 ④ 불변식: L over C ≈ I (선 영역 외 정확 일치)
```

### 1.2 2-pass 부트스트랩 (파라미터 자기 측정)

직조 주기 검출 → RGF → 1차 선 추출 → 선폭 분포 측정 → scale-space/클리핑 파라미터 확정 → 2차(최종) 선 추출.

### 1.3 파라미터 유도 (P-adapt 준수표)

| 파라미터 | 유도 방법 |
|---|---|
| RGF σ_s | 2D autocorrelation 직조 주기 |
| RGF 반복 종료 | 반복 간 변화량 중앙값 < σ_n (고주파 잔차 MAD로 추정한 노이즈 바닥) — 외부 상수 없음 |
| RGF σ_r | 그래디언트 크기 중앙값에서 유도 |
| 선 검출 스케일 | 측정된 선폭 분포 [5th, 95th] 백분위를 커버하는 scale-space, 샘플링 간격은 IQR에서 유도 (단일 k=1.6 의존 제거, 다중 굵기 대응) |
| 임계화 | Otsu 또는 백분위 히스테리시스 |
| 선폭 클리핑 | 선폭 분포 백분위 (v1의 고정 1~8px 폐기) |
| 스켈레톤 최소 길이 | 이미지 대각선 비율로 정규화 |
| Telea 확산 반경 | 국소 선폭 (폭 지도 재사용, 별도 상수 없음) |

### 1.4 자가 경쟁 게이트 (임계값 없는 fallback)

몰골법·발묵 화풍 대응. 신뢰도 임계값 대신 **경로 자가 경쟁**:

```
① 다운샘플 프록시(1/4 해상도)에서 두 경로 실행:
   경로 A: 분해 → C 직물 제거 → 재합성 / 경로 B: v1 통합 처리
② 품질 범함수 Q = (직조 대역 잔차 에너지 ↓) + (선 보존도: 처리 전후 선 응답 상관 ↑)
③ argmax 경로를 전체 해상도에 적용 — 임계값 없음, v1이 안전망
```

### 1.5 실패 모드

| 실패 모드 | 대응 |
|---|---|
| 직조 실의 선 오검출 | RGF 사전 억제 + 스켈레톤 최소 길이 필터 |
| 몰골법 (선 거의 없음) | 자가 경쟁 게이트 → v1 경로 |
| 선 제거 후 C의 색 번짐 잔재 | 얇은 선폭 + 국소 확산이라 영향 미미, 재합성 잔차로 검증 |

## 2. Stage 1: 색 레이어 직물 제거 심화

### 2.1 구조적 이득

v1 Algorithm 1의 타협 장치(α_min=0.3 부분 감쇠, r_axis/r_cross 분리, 윤곽 8px 보호 대역)는 모두 "직물 주파수와 선 고주파의 겹침" 때문이었다. 입력이 선 없는 C이므로 **타협 자체가 불필요**하다.

### 2.2 파이프라인

```
C 레이어
 ① 전역 격자 추정: 2D autocorrelation → 직조 기본 주기·방향(기저 벡터 2개)
    - v1 축 정렬 가정 폐기 (스캔 회전·경위사 비대칭 대응)
    - 고조파까지 격자에서 예측 → Fourier 피크 후보
 ② 피크별 위상 결맞음 검사 (신규 판별 원리):
    직물 = 전역적 위상 결맞음 주기 신호 / 내용물 = 위상 비결맞음
    → 패치별 피크 위상이 전역 격자와 일치할 때만 감쇠
 ③ 적응 notch 보간:
    notch 크기 = 피크별 2D Gaussian 피팅 실측 폭 (고정 annulus [2,5] 폐기)
    보간 링 = 국소 노이즈 바닥에서 유도 / 감쇠 = 완전 보간 (α_min 폐기)
 ④ 잔차 검증 루프: 직조 대역 잔차 에너지 < σ_n 까지 반복
 ⑤ NLM 단순화: h = f(σ_n) (고정 h=15 폐기), 보호 대역 폐기
L 레이어: 대비 정규화 (v1 contour enhancement의 이관 — boost=10 등 간접 강화 폐기)
→ 복원 이미지 R = L over C
```

### 2.3 v1 대비 변경 요약

| v1 장치 | v2 처리 |
|---|---|
| patch 64 / overlap 0.5 | 패치 크기 = 직조 주기 × 8 이상으로 유도 |
| r_axis=3 / r_cross=1 | 실측 격자 기저 + 피크별 Gaussian 폭 |
| annulus [2,5] | 피팅 폭 + 노이즈 바닥 |
| α_min=0.3 | 완전 보간 |
| 75th percentile 피크 선별 | 격자 예측 + 위상 결맞음 |
| NLM h=15, 보호 대역 8px | h=f(σ_n), 대역 폐기 |
| Contour boost=10 | L 대비 정규화로 이관 |

### 2.4 실패 모드

| 실패 모드 | 대응 |
|---|---|
| 캔버스 왜곡으로 주기 공간 변화 | 전역 격자 초기값 + 패치별 국소 정련 |
| 준법 등 반복 붓질의 결맞음 오인 | 결맞음 판정을 "전역 격자와의 일치"로 한정 |
| Telea 잔재의 스펙트럼 오염 | 저주파·비주기적이라 직조 대역과 분리, 잔차 루프에서 확인 |

## 3. Stage 3: Structure-first SSEI 2.0 (핵심 기여)

### 3.1 설계 철학

v1 SSEI는 "어디서 색을 가져올까"(C1–C4)에 집중했다. v2는 순서를 뒤집는다: 선을 먼저 완성하면 색 채움이 "작고 색-동질적인 영역 여러 개"로 분해된다.

### 3.2 Phase A — 선 완성 (L 레이어)

```
① 끊김 검출: 가림 경계의 스켈레톤 endpoint (방향, 곡률, 선폭, 농도 서술자)
② 전역 매칭: Hungarian assignment
   비용 = 연결 곡선 곡률 에너지 + 폭 불일치 + 농도 불일치
   "자연 종결(taper)"을 가상 매칭 상대로 포함 → 연결 vs 종결이 비교로 자동 결정 (임계값 없음)
③ 연결 곡선: Euler spiral(클로소이드), 양 끝 접선·곡률 G2 연속, 폭·농도 선형 보간
④ 검증: 완성 선이 기존 선과 교차하면 매칭 기각 → 재할당
```

v1 dynamic edge morphology(스켈레톤 + KD-tree + 2px fallback)는 Phase A의 특수 사례로 흡수 — 교차 경계선에서 내부 윤곽선 전체로 일반화.

### 3.3 Phase B — 영역 분할 색 채움 (C 레이어)

```
① 완성된 선이 가림 영역을 분할 → 각 조각은 인접 가시 영역의 연장
② 조각별 exemplar pool (C1 강화판): d_max 거리 제약 + 선 위상 제약
   (선을 넘지 않고 도달 가능한 픽셀만) → pool이 구조적으로 색-동질적
③ 채움 엔진: PatchMatch ANN (propagation + random search)
   + Wexler patch voting (겹침 패치 가중 평균) + multi-scale coarse-to-fine
④ C2(outside-in) = 스케일별 채움 순서로 유지
   C3(색 필터) = pool이 동질적이므로 안전망으로 강등
   C4(δ_safe) = 유지
```

### 3.4 Phase C — 재합성과 "by construction" 논증

L(완성) + C(채움) 알파 합성. **선 = 실측 기하의 보간(새 텍스처 생성 없음), 색 = exemplar 픽셀의 볼록 결합(voting)만** → post-fill smoothing 불필요 → v1 Eq 9와 smoothing(7단계)의 긴장 관계가 설계 수준에서 해소되고, 스타일 보존이 문자 그대로 by construction이 된다.

### 3.5 파라미터 유도

| 파라미터 | 유도 |
|---|---|
| 패치 크기 | C의 텍스처 자기상관 길이 |
| multi-scale 레벨 수 | log₂(결손 크기 / 패치 크기) |
| d_max | v1 해상도 적응 유지 + pool 미달 시 자동 확장 (필요 패치 수는 패치 크기에서 유도) |
| 매칭 수락 | 종결 비용과의 argmax 비교 |

### 3.6 실패 모드

| 실패 모드 | 대응 |
|---|---|
| 잘못된 선 연결 | Hungarian 전역 최적화 + 교차 검증 기각, 잔여 오류는 한계 명시 |
| 선으로 둘러싸인 고립 조각 | 경계 선 반대편 색 통계로 유사 조각 pool 차용 → 실패 시 전역 pool |
| 결손 내 미지 구조 (가시 힌트 전무) | 원리적 한계 (v1 동일), 한계 절 명시 |
| 몰골법 객체 | Stage 0 자가 경쟁이 v1 SSEI 경로 선택 |

## 4. 지원 스테이지 (Stage 2, 4) + 데이터 흐름

### 4.1 Stage 2 — 세그멘테이션

v1 구조(LabelMe + SAM 제약 정련) 유지, 부품 교체:

| 항목 | v1 | v2 |
|---|---|---|
| SAM 백본 | SAM vit_h | HQ-SAM (checkpoint 교체) |
| 2-pass + Eq 2-3 | 유지 | 유지 |
| 경계 처리 | distance transform feathering | closed-form alpha matting (선염 경계 충실) |
| layer_order | annotation 규약 | 유지 |

입력은 복원 이미지 R. 마스크는 좌표 기반이라 L·C 공용.

### 4.2 Stage 4 — 3D 복원

- 주 모델: **Art3D** (평면 일러스트/회화 특화, training-free) — v1 InstantMesh(실체 OpenLRM proxy)의 형상 환각 대응
- 비교군: TRELLIS / Hunyuan3D 2.5 / SF3D (정량 비교 표)
- 분해의 이득: L의 선명한 선 텍스처 → 3D 텍스처 윤곽 품질, alpha matte를 실루엣으로 전달
- 객체별 병렬화 추가 (P3의 귀결, v1 부재)

### 4.3 전체 데이터 흐름

```
원본 I
 ├─ Stage 0  분해 (L, C) ── 자가 경쟁 게이트 ──→ 탈락 시 v1 경로
 ├─ Stage 1  C: 격자→결맞음→notch→잔차루프→NLM / L: 대비 정규화 → R = L over C
 ├─ Stage 2  R에서 LabelMe → HQ-SAM 2-pass → alpha matting + layer_order → 가림 그래프
 ├─ Stage 3  객체별: Phase A 선 완성 → Phase B 영역 채움 → Phase C 재합성 → RGBA
 └─ Stage 4  객체별 병렬: Art3D → 360° 메시 (.glb)
```

### 4.4 스테이지 간 계약

| 경계 | 전달물 | 불변식 (기계 검증 가능) |
|---|---|---|
| 0→1 | L(RGBA), C(RGB), 직조 격자 | L over C ≈ I (선 영역 외 정확 일치) |
| 1→2 | R(RGB), σ_n | 직조 대역 잔차 < σ_n |
| 2→3 | 객체별 alpha matte + layer_order 그래프 | 마스크 좌표 L·C 공용 |
| 3→4 | 객체별 RGBA | 채움 픽셀 = exemplar 볼록 결합만 |

측정 통계(격자, σ_n, 선폭 분포)는 Stage 0-1에서 1회 측정 후 공유. P3 유지: 객체 실패 비전파.

## 5. 검증·실험 설계 (정량 전용)

### 5.1 기계 검증 (불변식의 자동 테스트화)

| 불변식 | 검증 |
|---|---|
| L over C ≈ I | 선 영역 외 완전 일치 + 선 영역 재합성 잔차 |
| 직조 잔차 < σ_n | 처리 후 격자 위치 스펙트럼 에너지 |
| by construction | 채움 픽셀이 조각 exemplar 볼록 결합 범위 내인지 픽셀 검사 — v1은 주장, v2는 기계 증명 |
| 선 G2 연속 | 연결점 곡률 불연속 측정 |

### 5.2 합성 GT 실험 (정량의 뼈대)

```
프로토콜 A (직물): 직조 없는 도판 + 실측 직조 오버레이(주기·방향·강도 변조)
                   → 제거 후 PSNR/SSIM/LPIPS
프로토콜 B (가림): 완전한 객체 + 인공 가림(크기·형상 변조)
                   → 복원 후 원본 비교 + 선 완성 스켈레톤 IoU
```

크기·형상·화풍 축의 성능 곡선으로 동작 한계를 정직하게 제시.

### 5.3 Ablation 매트릭스

| Ablation | 입증 대상 |
|---|---|
| 분해 on/off (v2 vs v1) | 백본의 가치 (주 기여) |
| 위상 결맞음 on/off | Stage 1 신규 판별 원리 |
| Phase A on/off | structure-first의 가치 (주 기여) |
| voting vs 단일 픽셀 복사 | smoothing 제거의 근거 |
| multi-scale on/off | 대형 결손 대응 |
| 자가 경쟁 vs 항상 분해 | 화법별 게이트 동작 (층화 데이터 필요) |

### 5.4 비교군

- v1 (동일 데이터·지표, 가장 중요)
- Criminisi, PatchMatch 원본 (training-free 동급)
- LaMa, SD-inpainting (학습 기반 상한 대비 격차 — 리뷰어 필문)

### 5.5 스타일 정량 지표군 (정성 평가 대체)

원리: **채움 영역과 가시 영역의 내부 통계가 구별 불가능해야 한다.**

| 지표 | 측정 | 성격 |
|---|---|---|
| SIFID | 채움 vs 가시 영역 패치 분포 거리 | 단일 이미지 내부 통계, 사전학습 추론만 |
| Gram matrix 거리 (VGG) | 스타일 표현 차이 | 스타일 전이 문헌 표준 — Eq 9 직결 |
| 패치 통계 발산 | 색/gradient/wavelet 분포의 KS·EMD | 순수 통계 (딥러닝 무관) |
| 선 통계 일치도 | 완성 선 vs 원본 선의 선폭·곡률 분포 KS | Phase A 전용 |
| No-reference IQA | MUSIQ, CLIP-IQA | 쌍대 비교 자동화 |

by construction 설계와 맞물려 색 히스토그램·Gram 거리가 구조적으로 작아야 하므로, 지표가 설계 검증을 겸하며 v1(smoothing이 새 색 생성) 대비 격차가 주 기여의 정량 증거가 된다.

한계 절 서술: "지각적 품질의 최종 판단은 인간 평가가 이상적이나, 본 연구는 채움 영역과 가시 영역의 통계적 구별 불가능성(SIFID, style distance)을 스타일 보존의 조작적 정의로 채택한다."

## 6. 논문 기여 서술 (정리)

1. **구륵법 인지 선·색 분해 백본**: 하나의 도메인 표현이 복원(Stage 1)과 인페인팅(Stage 3)을 동시에 단순화
2. **위상 결맞음 직물 판별**: 임계값 기반 피크 선별을 물리적 판별 원리로 대체
3. **Structure-first SSEI 2.0**: 선 완성(Euler spiral + Hungarian + 비교 기반 종결) → 영역별 exemplar 채움(PatchMatch ANN + voting + multi-scale)
4. **강화된 by construction 스타일 보존**: 기계 검증 가능한 형태로 승격 (post-fill smoothing 폐기)
5. **P-adapt**: 튜닝 상수 0개 (이미지 통계 유도 + argmax 자가 경쟁)
6. **자가 적응 화법 대응**: 몰골법에서 시스템이 스스로 v1 경로 선택

## 7. 한계 (선제 명시)

- 결손 내부의 완전한 미지 구조는 복원 불가 (self-exemplar 원리적 한계, v1 동일)
- 선 연결의 의미적 오류(다른 획 연결)는 기하 비용만으로 완전 배제 불가
- 학습 기반 인페인팅 대비 절대 품질 상한 존재 가능 — training-free의 대가로 명시
- 스타일 보존의 조작적 정의(통계적 구별 불가능성)는 인간 지각과 완전 일치하지 않을 수 있음
